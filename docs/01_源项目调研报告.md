## 1. 项目概况

| 项目 | 内容 |
|---|---|
| Unity 版本 | `2020.3.49f1c1`（`ProjectSettings/ProjectVersion.txt`） |
| 渲染管线 | **URP 10.10.1**（`com.unity.render-pipelines.universal`）；额外有 2 个雾效着色器是 Built-in RP（CG） |
| 资源根目录 | `Assets/IL3DN/`（Illustrated 3D Nature 缩写） |
| 场景数 | 4 个（Demo、PineForest、PineForest_Winter、Assets 展示厅） |
| 源代码总量 | 11 个运行时脚本 + 5 个 Editor 脚本 ≈ **1396 行** |
| 着色器 | 15 个，全部为 **Amplify Shader Editor 导出的手写 HLSL/CG**（无 Shader Graph、无 `.cginc`） |
| 总磁盘占用 | Audio 91M、Meshes 14M、Textures 68M（合计约 173M 资源） |

> **风格定位**：Illustrated（插画式）写实风格自然场景。低饱和度、无金属/无光滑、纯色 tint + cutout alpha、程序化 simplex noise、梯度天空、深度雾、全局雪/风系统。整套视觉效果通过**减法**而非加法实现（移除法线贴图、金属、高光），不是自定义着色模型。

## 3. 脚本清单（运行时，共 11 个）

所有脚本位于 `IL3DN` 命名空间。详细字段与交互见 `docs/03_技术规格_脚本与Shader.md`，这里给出职责速览。

| 脚本 | 基类 | 职责 |
|---|---|---|
| `IL3DN_SimpleFPSController` | MonoBehaviour | 第一人称移动/视角/跳跃/重力/脚步声；`[RequireComponent(CharacterController, AudioSource)]` |
| `IL3DN_SimpleMouseLook` | **纯 Serializable 类** | 鼠标视角（pitch clamp + 光标锁定），作为 FPSController 的子字段 |
| `IL3DN_ChangeWalkingSound` | MonoBehaviour | 数据标记组件（trigger collider 上），切换玩家脚步/跳/落地音效 |
| `IL3DN_PlayRandomSound` | MonoBehaviour | 周期性随机播放环境音（鸟叫等） |
| `IL3DN_ColorController` | MonoBehaviour | 仅持有 `int slot`；被两个 ColorManager 通过 `RequireComponent` 自动挂载 |
| `IL3DN_ColorManagerEffects` | MonoBehaviour | 多颜色属性的季节调色板切换（针对 `_NearColor/_FarColor/_GlowColor` 等） |
| `IL3DN_ColorManagerTextures` | MonoBehaviour | 颜色+贴图的季节调色板切换（`material.color` + `mainTexture`） |
| `IL3DN_Wind` | MonoBehaviour | `[ExecuteInEditMode]` 全局风驱动器（风向/强度/速度/扰动 + 叶子草摆动） |
| `IL3DN_Snow` | MonoBehaviour | `[ExecuteInEditMode]` 全局雪驱动器（各类表面雪量 + alpha cutoff） |
| `IL3DN_Fog` | MonoBehaviour | Built-in RP 全屏图像特效雾（`OnRenderImage` + `Graphics.Blit`） |
| `IL3DN_Fog_PP` | PostProcessEffectSettings | PostProcessing Stack v2 版雾（`#if UNITY_POST_PROCESSING_STACK_V2`） |

**5 个 Editor 脚本**：分别为 ColorController / ColorManagerEffects / ColorManagerTextures / Snow / Wind 提供 CustomEditor，主要是绘制装饰性 header 图片和分组。

### 全局着色器变量契约（必须严格保留）

`IL3DN_Wind.cs` 每帧 `Shader.SetGlobalXxx` 推送：

```
Texture2D  NoiseTextureFloat
Vector3    WindDirection          (= transform.rotation * Vector3.back)
float      WindStrenghtFloat      (注意拼写 "Strenght")
float      WindSpeedFloat
float      WindTurbulenceFloat
float      LeavesWiggleFloat
float      GrassWiggleFloat
Shader keyword: _WIND_ON, _WIGGLE_ON
```

`IL3DN_Snow.cs` 每帧推送：

```
float  SnowPinesFloat / SnowLeavesFloat / SnowBranchesFloat
float  SnowRocksFloat  / SnowGrassFloat  / SnowTerrainFloat
float  AlphaCutoffFloat
Shader keyword: _SNOW_ON
```

> 这些是脚本与着色器的唯一耦合点。**着色器必须读取同名 global 变量和 keyword，C# 才能驱动新写的 shader**。

### 已知的源代码"特性/坑"（移植时需注意）

1. `IL3DN_PlayRandomSound` 在 `SetupSound()` 立即重 roll 后才播放，导致"预选的 clip 永远不会用到"。
2. `PlayFootStepAudio` 用了 Standard Assets 的"换到 index 0"防重复洗牌法。
3. 故意拼错的公开字段名需保留以兼容 Inspector 序列化：`WindStrenght`、`m_RunstepLenghten`、`meterial`。
4. ColorManager 的 slot 是**外 1 内 0**（`SetMaterialColors(slot)` 内部 `slot - 1`）。
5. `IL3DN_ColorManagerEffects` 内 `MaterialColors` 构造函数依赖 `UnityEditor.ShaderUtil`，仅在编辑器下能枚举颜色属性。

### 视觉风格要点（"插画感"如何实现）

1. **统一 tint 贴图**：几乎全是 `_Color * _MainTex`；地形是每层 `_ColorN * _SplatN`。
2. **无金属、无高光、无法线贴图**（除水）。法线固定 `(0,0,1)`，颜色完全来自光照与几何法线。
3. **低饱和度色板**：松皮棕 `(0.594,0.407,0.305)`、水底紫 `(0.278,0.096,0.415)`、水腰青 `(0.114,0.529,0.454)`、天空青→板岩蓝渐变。
4. **cutout alpha 遍地**：叶/卡/草/云/松。
5. **程序化 simplex noise**：用于贴花、水、风扰动，替代细节贴图。
6. **梯度天空盒**：双 HDR 色沿世界 Y。
7. **基于世界法线 Y 的雪累积**：每类资产一个独立雪量 global（`SnowBranchesFloat` 等），全局一键换季。
8. **风是唯一动画**：世界空间 noise 平移 + 顶点色权重（`.g`=叶子/上层权重，`.a`=风权重）。
9. **深度雾**：近/远/发光色三段染色。
10. **软粒子**：深度 fade 避免粒子穿模。

### 顶点色编码约定（建模必须遵守）

```
.g  = 叶子/上层权重（影响雪量与风）
.a  = 风权重（松/草/叶都用）
```

## 8. 关键风险点（移植到 Unity 6 时）

| 风险 | 说明 | 对策 |
|---|---|---|
| URP API 大版本变化 | 原 shader 用 `UniversalFragmentPBR`、`GetMainLight` 等 URP 10 API；Unity 6 是 URP 17，部分 API 签名/包含文件变化 | 着色器迁移到 URP 17 的 `URP/Includes`；用 Shader Graph 兼容性子目标或手改 include |
| Built-in 雾效失效 | `OnRenderImage` / PostProcessing Stack v2 在 Unity 6 URP 中不工作 | 改写为 **URP Full Screen Pass Renderer Feature**（或 Blit Renderer Feature） |
| `ShaderUtil` 颜色枚举 | ColorManagerEffects 编辑器侧枚举依赖旧 API | 用 `ShaderUtil` 的新签名或 `Shader.PropertyToID` 手维护属性列表 |
| `PostProcessEffectSettings` 类 | PPv2 在 Unity 6 已移除 | `IL3DN_Fog_PP` 整个类移除，逻辑迁到 Renderer Feature 的 settings |
| Legacy Input 轴 | FPSController 用 `Input.GetAxis("Horizontal")` 等 | Unity 6 仍兼容旧 Input Manager（可同时启用新旧），保留即可；若要换新 Input System 需重写控制器 |
